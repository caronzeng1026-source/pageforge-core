# WebEdit 产品功能汇总与使用说明文档

WebEdit 是一款功能强大的浏览器原生网页编辑器（WYSIWYG），允许用户实时修改任何网页的文本、样式和布局，并支持将修改后的结果导出。

---

## 1. 产品核心功能概览

### 基础编辑

- **实时文本编辑**：开启编辑模式后，双击页面任意文本即可进入编辑状态，支持即时修改。
- **元素高亮与选中**：鼠标悬停自动高亮元素边界，点击即可选中并呼出侧边栏属性面板。
- **撤销与重做 (Undo/Redo)**：支持多达 50 步的操作历史记录，随时回退错误操作。

### 样式与视觉调整

- **全方位 CSS 编辑**：通过可视化面板调整字体、颜色、背景、边框、透明度、圆角等。
- **间距管理**：精细控制元素的内边距 (Padding) 和外边距 (Margin)。
- **高级缩放**：选中元素后支持通过 8 个方向的缩放手柄手动调整尺寸。

### 布局与结构增强

- **智能拖拽 (Drag & Drop)**：
  - **位置移动**：自由拖动元素至父容器或相邻位置。
  - **并排放置 (Wrap Drop)**：将元素拖动至目标元素的左/右侧边缘，系统将自动创建一个 Flex 容器包裹两者，实现并排布局。
- **布局模式切换**：支持 Block、Flex、Grid 三种核心布局模式，并提供详细的子项控制（如对齐方式、间距、换行等）。
- **多选编组**：支持 `Shift/Cmd/Ctrl + 点击` 选中多个元素，并可一键将其打包为水平或垂直排列的容器。

### 交互与体验

- **历史回溯滑块 (History Slider)**：独创的“时空穿梭”功能。开启对比预览后，可通过滑块平滑地查看从原始网页到最新修改间的每一个中间状态。
- **响应式视口切换**：一键切换桌面 (Desktop)、平板 (768px) 和移动端 (390px) 视图。
- **多主题界面**：侧边栏支持深色、浅色、玻璃态、赛博朋克等多种审美主题。

## 2. 使用说明指南

### 启动编辑器

1. 点击浏览器右上角的 WebEdit 插件图标，打开侧边栏界面。
2. 点击侧边栏顶部的 **“开启编辑模式”** 按钮。

### 修改网页内容

- **修改文字**：双击任一文字块，修改后点击空白处或按 `Enter` 保存，按 `Escape` 取消。
- **修改样式**：点击选中元素，在侧边栏的“编辑”标签页中调整各项参数。
- **移动元素**：长按已选中的元素并拖动，根据页面出现的蓝色指示线确定放置位置。
- **删除/隐藏**：选中元素后，点击侧边栏下方的“删除”或“隐藏”按钮，或使用快捷键。

### 插入新元素

1. 切换到侧边栏的 **“插入”** 标签页。
2. 选择要添加的元素（如文本块、按钮、标题、色块或 Flex 布局容器）。
3. 元素将根据当前选中的位置自动插入。

### 对比与回退

- 点击顶部的 **“对比原文档”**（眼睛图标）进入对比预览。
- 使用弹出的滑块左右拖动，观察修改前后的视觉差异。

---

## 3. 导出与保存

WebEdit 提供了三种导出方式，满足不同开发需求：

1. **导出完整 HTML**：获取当前的完整 DOM 结构，包含所有内联样式修改。
2. **CSS Patch**：仅导出您所做的样式修改，生成基于唯一 ID 选择器的 CSS 规则文件，方便集成到现有项目。
3. **Action Log**：导出包含所有操作步骤的 JSON 日志，用于审计或二次开发。

## 4. 键盘快捷键 (Hotkeys)

| 快捷键 | 功能说明 |
| :--- | :--- |
| **Ctrl/Cmd + Z** | 撤销上一步操作 |
| **Ctrl/Cmd + Shift + Z / Ctrl + Y** | 重做操作 |
| **Ctrl/Cmd + C** | 复制选中元素（支持多选复制） |
| **Ctrl/Cmd + V** | 粘贴元素（智能选择插入位置） |
| **Delete / Backspace** | 删除选中的元素 |
| **Escape** | 取消当前选中 / 取消正在进行的拖拽 |

## 5. 高级技巧

> [!TIP]
> **如何快速制作并排对比布局？**
>
> 1. 先准备好两个元素。
> 2. 选中其中一个，按住并拖动到另一个元素的左侧或右侧边缘。
> 3. 当看到垂直的蓝色指示线时松开，系统会自动创建 Flex 容器并将它们等宽分布。

> [!IMPORTANT]
> **关于响应式预览**
> 在视口切换模式下，页面将模拟不同设备的尺寸。这对于检查响应式断点（Breakpoints）和移动端适配非常有帮助，且在此模式下依然可以继续编辑。
